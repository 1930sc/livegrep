This live-grep project was an experiment in scaling real-time regex search, inspired in part by the death of Google Codesearch.

live-grep uses Russ Cox's excellent RE2 regular expression library, underneath a custom indexing and searching backend written in C++ with pthreads. The frontend is implemented in node.js using socket.io for the realtime communication with the browser.

livegrep has been tested on recent Chrome and Firefox, as well as IE8 and 9. I have reports of it working in recent Safari, as well. Bug reports are appreciated, either via the "feedback" button above, or via email.
